A modern, clean, and highly performant personal portfolio website built with a focus on semantic HTML and scalable styling.

##  Tech Stack
*   **HTML5**: Clean, semantic markup for better SEO and accessibility (A11y).
*   **SCSS**: Organized styling using variables, nesting, and mixins for maintainability.
*   **BEM Methodology**: Consistent naming conventions to ensure a scalable CSS structure.
